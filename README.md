Schliesst ein Akkordeon Artikel-übergreifend.  
Beim Öffnen eines Akkordeons in einem z.B. Artikel-3, wird das zuvor geöffnete Akkordeon - z.B. aus Artikel-1 - geschlossen.  

Das Twig-Template wird pro Artikel, pro Akkordeon-Inhaltselement ausgewählt.  
Es kann aber auch global über die `/templates/content_element/accordion.html.twig` gesteuert werden.
